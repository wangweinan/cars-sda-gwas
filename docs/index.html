Method

Three innovations combined into a single pipeline

1. Jin-Cai Empirical Null

Estimate N(μ₀, σ₀²) from the characteristic function at moderate frequencies. No arbitrary tuning or central-region selection. Accurately recovers σ₀ = 1.17 (true: 1.16) in simulation.

2. CARS Density Ratio

Compute local FDR conditional on MAF via per-bin EM mixture with the empirical null as the fixed null component. The density ratio f₀(z|s)/f(z|s) captures covariate-dependent signal enrichment.

3. K-Fold Cross-Fitting

Prevent circularity: density parameters are estimated on training folds and applied to held-out test folds. Maintains the independence assumption needed for valid FDR control.

Quick Start

Minimal Python code to run CARS-SDA on your own GWAS data

from src.cars_sda import cars_sda

# Your GWAS summary statistics
Z = beta / se              # z-scores
S = minor_allele_freq      # auxiliary covariate

# Run CARS-SDA (auto-estimates empirical null)
rejections, lfdr, threshold, mu0, sigma0 = cars_sda(Z, S, alpha=0.05)

print(f"Discoveries: {rejections.sum():,}")
print(f"Empirical null: N({mu0:.3f}, {sigma0:.3f}²)")
