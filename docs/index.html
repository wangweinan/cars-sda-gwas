The CARS Density-Ratio Statistic
CARS(i) = f₀(Z) · (|Tτ|/m) · f*(S | Tτ) / (correction · f(Z, S))

f₀ = Jin-Cai empirical null on LDSC-adjusted Z  ·  f(Z,S) = 2D FFT-KDE  ·  S = MAF covariate  ·  correction = MC P(lfdr≥τ|H₀)

Why the +11.7% metric matters more than the fake "+235%"

Comparing CARS (+235%) against an archaic BH-GC baseline is a strawman. BH-GC destroys true polygenicity. The fair, rigorous baseline is Adaptive-Z, which uses the exact same LDSC+JinCai null estimation as CARS. Our +11.7% locus gain (175 strictly novel loci) represents the pure algorithmic benefit of bivariate joint density estimation (exploiting covariate structure) over univariate thresholding.

⚠ Independence assumption

CARS assumes independent tests. LD violates this. Formal FDR control requires SDA (Symmetrized Data Aggregation; Du et al. 2023) — not yet implemented. The step-up procedure likely holds under PRDS, but independent replication is needed. MHC (chr6:25-35Mb) is clean. 1 SNP flagged in 8p23.1 inversion and excluded.

Methodology

Four innovations combined · LDSC-corrected for confounding · Jin-Cai for polygenic null

① LDSC Confounding Adjustment

Divide raw Z-scores by √(LDSC intercept) to remove population stratification before fitting the empirical null. Preserves true polygenic signal.

Z_adj = Z_raw / √1.0664 → σ₀ shrinks from 1.21 → 1.14

② Jin-Cai Empirical Null

Fourier-based estimation of the true null N(μ₀, σ₀²) from the empirical characteristic function. σ₀ = 1.144 captures the residual polygenic background.

φ(t) = E[e^{itZ}] → log|φ|² ~ a + b·t² → σ₀² = −b

③ Bivariate CARS Density Ratio

Joint density f(Z, MAF) estimated via 2D FFT-KDE in O(n + G²·log G). Exploits covariate information to lower thresholds where power is highest.

CARS = f₀(Z) · f*(S|T_τ) / (correction · f(Z,S))

④ Barber-Candès Step-Up

FDR control via mirror-statistic step-up at α=0.05. Monte Carlo correction for screening-set bias. Note: not SDA — assumes independence.

correction = P(lfdr ≥ 0.9 | Z ~ N(μ₀, σ₀²)) via 50K MC

Quick Start

Run CARS-JC v4.0 on your own GWAS data

from src.cars_sda import cars_sda, jin_cai_empirical_null
import numpy as np

Z = beta / se              # raw z-scores
S = minor_allele_freq      # auxiliary covariate (MAF)

# Step 1: LDSC confounding adjustment
ldsc_intercept = 1.0664   # from ldsc --h2
Z_adj = Z / np.sqrt(ldsc_intercept)

# Step 2: Bivariate CARS with Jin-Cai null
rej, cars_stat, thr, mu0, sigma0, diag = cars_sda(Z_adj, S, alpha=0.05)
print(f"Discoveries: {rej.sum():,}")
print(f"Empirical null: N({mu0:.3f}, {sigma0:.3f}²)")
