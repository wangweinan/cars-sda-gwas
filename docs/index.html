Why locus-level, not variant-level?

Variant-level counts are inflated ~127× by linkage disequilibrium. A single causal variant creates ~127 significant neighbors. We report locus-level discoveries (500kb distance-clumped) for scientific accuracy. CARS gains +14.3% independent loci — the bivariate f(Z, MAF) captures real signal enrichment at common allele frequencies.

⚠ Theoretical caveat

CARS (Cai et al. 2019) assumes independent tests. GWAS LD violates this. FDR control likely holds under PRDS conditions, but locus-level FDR requires post-hoc clumping. The 371 net new loci are at suggestive significance (p ~ 10⁻⁴) and require independent replication.

Methodology

Four innovations combined into a single pipeline

① Jin-Cai Empirical Null

Estimate the true null N(μ₀, σ₀²) from the empirical characteristic function at moderate frequencies. Corrects genomic inflation (σ₀ = 1.098, λ_GC = 1.34) without arbitrary central-region selection.

φ(t) = E[e^{itZ}] → log|φ|² ~ a + b·t² → σ₀² = -b

② Bivariate CARS Statistic

The CARS density ratio uses the joint density of (Z, MAF) — not just conditional binning. The screened set Tτ estimates the null covariate distribution.

CARS = f₀(Z) · f*(S|T_τ) / (correction · f(Z,S))

③ 2D FFT-KDE

Product-kernel KDE via 2D FFT convolution. O(n + G²·log G) — handles 37.6M variants in seconds. Silverman bandwidth per dimension.

f̂(z,s) = IFFT₂(FFT₂(counts) · FFT₂(K₁⊗K₂)) / n

④ Monte Carlo Correction

Calibrate the screening set by computing P(lfdr ≥ τ | H₀) from 50K null Monte Carlo draws. Prevents optimistic bias from using data-dependent screening.

correction = P(lfdr_marginal ≥ 0.9 | Z ~ N(μ₀,σ₀²))

⚠️ Colocalization Note

Coloc-ABF analysis yielded PP.H4 ≈ 0 for all 17 GTEx loci. This is expected for sub-genome-wide-significant hits (p~10⁻⁴): the GWAS Bayes factor is modest while eQTL evidence is overwhelming, causing H2 (eQTL-only) to dominate. Coloc was designed for GWS variants (p<5×10⁻⁸). For our sub-threshold discoveries, cross-study concordance (89.8%) and sc-eQTL coverage (11.6%) are more appropriate validation metrics.

Quick Start

Run CARS-SDA v3.0 on your own GWAS data

from src.cars_sda import cars_sda

Z = beta / se              # z-scores from GWAS
S = minor_allele_freq      # auxiliary covariate (MAF)

# Proper bivariate CARS with 2D FFT-KDE
rej, cars_stat, thr, mu0, sigma0, diag = cars_sda(Z, S, alpha=0.05)
print(f"Discoveries: {rej.sum():,}")
print(f"Empirical null: N({mu0:.3f}, {sigma0:.3f}²)")
